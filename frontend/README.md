# PeerPrep v0.1

PeerPrep is all-in-one place where students can find peers to practice whiteboard-style interview questions together.

![Ui](../frontend/public/images/UI.png)

This project uses the following tech stack:

### Frontend
1. Vite
2. React
3. TailwindCSS

### Backend
1. MongoDB
2. Javascript
3. NodeJs
4. ExpressJs
